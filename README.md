# ProfileSync AI

Plataforma de gestão e geração assistida de perfil profissional Dev.

O ProfileSync AI tem como objetivo centralizar projetos, tecnologias, experiências práticas e evolução técnica, permitindo gerar materiais profissionais com apoio de IA de forma estruturada, ética e reutilizável.

---

## Objetivo

Transformar evolução técnica e experiência prática em posicionamento profissional organizado e reutilizável.

A plataforma busca auxiliar desenvolvedores na construção e manutenção de:

- Currículo Dev
- Resumo profissional
- Bio para LinkedIn
- Descrição de projetos
- Histórico técnico
- Posicionamento profissional

---

## Funcionalidades Planejadas

- Cadastro de projetos
- Cadastro de tecnologias
- Registro de entregas técnicas
- Geração assistida de currículo
- Geração de resumo profissional
- Geração de descrição para GitHub
- Exportação em Markdown/PDF
- Histórico de evolução profissional
